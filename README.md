# COMP5123M Coursework 2 2026

Repo for codes, notes and resources related to the COMP5123M Coursework 2 2026.

`safety-service.yaml`: Service config for AdGuard
`safety-vnf.yaml`: DEployment config for AdGuard
`prometheus.yml`: Local config for running prometheus on local machine
`dns_list.txt`: The DNS list used for testing AdGuard
`node-exporter.yaml`: The File for deploying node-exporter

The notes folder contains generic md files I used for keeping copies of commands I used. These files are incomplete as a significant amount of debugging that was done I forgot to retain.

DNS perf logs for the testing can be found in `k3s_test_dnsperf.txt` and `minikube_test_dnsperf.txt`
